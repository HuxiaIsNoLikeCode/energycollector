能量收集者 — 玩家手册
============

> cccccc | 版本：v1.0.8

* * *

游戏简介
----

你是一个悬浮在赛博空间中的光球，不断有能量粒子从上方飘落。左右倾斜手腕控制光球移动，收集能量粒子获取积分和能量，同时躲避红色障碍物。存活越久、收集越多，得分越高。

* * *

操作方式
----

| 操作   | 方式            |
| ---- | ------------- |
| 左右移动 | 手腕向左/右倾斜      |
| 释放技能 | 点击屏幕右下角「技能」按钮 |

游戏过程中屏幕保持常亮，不会自动息屏。

* * *

界面说明
----

    ┌──────────────┐
    │ 分数: 120  Lv.2  ← 当前分数 & 等级
    │ E: 85           ← 本局获得的能量
    │              │
    │   ●          ← 能量粒子（各种颜色）
    │     ◆        ← 障碍物（红色系，菱形）
    │              │
    │     ◉        ← 你的光球（青色，屏幕中央）
    │              │
    │          [技能] ← 技能按钮
    └──────────────┘

* * *

能量粒子
----

游戏中有 4 种能量粒子，从上方飘落：

| 颜色  | 基础分值 | 出现概率 | 说明       |
| --- | ---- | ---- | -------- |
| 青色  | 10   | 50%  | 最常见，稳定收入 |
| 紫色  | 20   | 30%  | 中等价值     |
| 黄色  | 50   | 15%  | 高价值，优先收集 |
| 绿色  | 30   | 5%   | 稀有，价值较高  |

> **注意**：实际分值会根据难度和升级倍率动态调整（见「难度系统」章节）。

关闭「粒子效果」后，粒子生成频率大幅降低且尺寸变小（约正常 30%），游戏更简洁流畅。

* * *

难度系统
----

游戏提供三档难度，影响多个维度：

| 参数      | 简单        | 困难       | 地狱        |
| ------- | --------- | -------- | --------- |
| 障碍物速度   | 0.6x 慢    | 1.4x 快   | 1.8x 很快   |
| 障碍物数量   | 较少        | 较多       | 很多        |
| 粒子数量    | 多         | 少        | 极少        |
| 粒子大小    | 大         | 小        | 极小        |
| 粒子/能量倍率 | 0.8x (低)  | 1.5x (高) | 2.5x (极高) |
| 碰撞判定    | 宽松（需很近才撞） | 严格       | 很严格       |

**选择建议**：

* **简单**：新手入门，轻松体验收集乐趣
* **困难**：高风险高回报，粒子少但每个分值高
* **地狱**：极限挑战，障碍密集、粒子稀缺但单个收益极高

* * *

障碍物
---

3 种障碍物以不同速度从上方飘落，碰到即 Game Over：

| 颜色  | 基础速度 | 特点    |
| --- | ---- | ----- |
| 红色  | 慢    | 最容易躲避 |
| 橙红  | 中    | 需要注意  |
| 粉红  | 快    | 最危险   |

> 实际速度 = 基础速度 × 难度系数（见上表）

* * *

等级系统
----

* 每获得 300 分自动升级
* 等级越高，粒子和障碍物出现越频繁
* 最高等级无上限

* * *

技能
--

点击右下角「技能」按钮，触发 **范围扩大** 效果：

* 持续 3 秒
* 拾取范围翻倍（屏幕上显示黄色透明圆圈表示范围）
* 冷却时间：默认 6 秒（可通过升级减少到最低 3 秒）

* * *

能量与升级
-----

收集的能量会**累计保存**，即使游戏结束也不会丢失。在主菜单的「能力升级」页面可以消耗能量升级 4 项属性：

| 属性   | 效果                | 升级费用     |
| ---- | ----------------- | -------- |
| 移动速度 | 手腕倾斜灵敏度 +0.5      | 100 能量/级 |
| 收集范围 | 自动拾取半径 +8px       | 150 能量/级 |
| 吸收效率 | 能量获取 +30%/级       | 200 能量/级 |
| 技能冷却 | 冷却时间 -1 秒/级（最低3秒） | 250 能量/级 |

> 升级数据实时持久化，退出重进不丢失。

* * *

设置
--

在主菜单的「设置」页面可以调整：

| 设置项  | 说明                       |
| ---- | ------------------------ |
| 振动反馈 | 开启/关闭拾取和碰撞时的手环振动         |
| 游戏难度 | 简单 / 困难 / 地狱             |
| 粒子效果 | 开启/关闭粒子特效（关闭后粒子稀疏且微小）    |
| 重置数据 | 清除所有进度（能量、最高分、升级等级），不可恢复 |

* * *

游戏结束

碰到障碍物后游戏结束，显示：

* **分数**：本局总得分
* **获得能量**：本局收集的能量
* **累计能量**：所有局次的总能量（已持久化保存）

可选择「重新开始」（保留累计能量继续玩）或「返回菜单」。

* * *

数据存储

以下数据会自动保存到手环本地存储，关闭应用后不丢失：

* 累计能量（每局累加）
* 最高分（取历史最高）
* 升级等级（4 项属性各自独立）
* 游戏设置（振动、难度、粒子效果）

> 所有数据使用本地 storage 直接读写，确保跨页面同步可靠。

* * *

小贴士

1. **简单模式**适合熟悉操作，**困难/地狱模式**虽然危险但单次收益更高
2. 优先收集黄色粒子，单个价值最高（尤其在地狱模式下 ×2.5 倍率）
3. 障碍物会跟随你的移动方向偏移，不要一直往一个方向躲
4. 技能冷却好就用，扩大范围可以同时收集更多粒子
5. 先升级「吸收效率」收益最高，每次收集的能量直接增加
6. 如果觉得卡顿或眼花，可以关闭粒子效果
7. 地狱模式下粒子极少但每个都很值钱，耐心等待并精准收集是关键
8. 「重新开始」不会丢失已获得的能量，放心大胆地连续挑战
