# 能量收集者 (Energy Collector)

> 适配设备：**小米手环 Pro 系列**（10 Pro 系列等） | 版本 v1.1.2

---

## 游戏介绍

你是一个悬浮在赛博空间中的光球，不断有能量粒子从上方飘落。左右倾斜手腕控制光球移动，收集能量粒子获取积分和能量，同时躲避红色障碍物。存活越久、收集越多，得分越高。

收集的能量可以用于升级 4 种能力（移动速度 / 收集范围 / 吸收效率 / 技能冷却），让后续挑战更加轻松。

游戏提供三档难度：**简单**（新手友好）、**困难**（高风险高回报）、**地狱**（极限挑战），满足不同玩家需求。

---

## 关于本项目

- 本项目 UI 仅适配 **小米手环 Pro 系列** 设备（1.8" AMOLED, 192×490px）
- 一个打发闲暇时光、顺便增加手环耗电量的**小游戏**
- 基于 Vela OS + QuickJS 开发，纯前端实现，无后端依赖
- 所有数据本地存储，断网也能玩
